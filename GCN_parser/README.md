### Google API Python Libraries
- **Packages Installed:**
  - google-api-python-client
  - google-auth-httplib2
  - google-auth-oauthlib
- **Method:** pip (command line)
- **Command Used:**
pip install --upgrade google-api-python-client google-auth-httplib2 google-auth-oauthli

- **Purpose:** Required for accessing and authenticating with Google APIs in Python.
- **Notes:** Installed with --upgrade to ensure latest versions.
